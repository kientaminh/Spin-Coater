# Spin Coater
Spin coater là máy phủ chất lỏng lên bề mặt phẳng. Đây là thiết bị được xây dựng ở lab AIoT PTIT.

## Phần cứng

- Động cơ 775
- Màn hình Nextion
- Esc480A
- Cảm biến Encoder
- Esp32
- Hệ thống hút chân không
